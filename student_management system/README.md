# Student Management System

A simple Student Management System built using Python.

## Features

- Add Student
- View Students
- Search Student by Roll Number
- Delete Student
- Stores data in a text file

## Technologies

- Python 3
- File Handling

## How to Run

```bash
python student_management.py
```

## Folder Structure

```
student-management-system/
│── student_management.py
│── students.txt
│── README.md
```
